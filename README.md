☕ Coffee Shop Mobile App UI/UX (Figma)

A modern and elegant coffee shop mobile application UI/UX design created using Figma.
This project focuses on delivering a smooth user experience with a visually appealing interface for coffee lovers.

🚀 Features

🔐 User Authentication (Login & Sign Up)

🏠 Home Dashboard with product listings

🔍 Search functionality

☕ Coffee product details page

🛒 Add to cart & cart management

📦 Order placement interface

❤️ Wishlist (favorites)

📱 Clean and responsive mobile UI design

🎨 Design Highlights

Warm coffee-themed color palette ☕

Minimal and modern layout

Smooth user flow from login → order

High-quality product visuals

User-friendly navigation

🛠️ Tools Used

Figma (UI/UX Design)

Icons & UI Components

Typography & Color System
